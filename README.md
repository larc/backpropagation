# backpropagation

This code is a simple implementation of the backpropagation neural networks algorithm in c++ using the library [Armadillo](http://arma.sourceforge.net/) for matrix operations.

## Build and Compile

    mkdir build
    cd build
    cmake ..
    maje -j 4
    ./backprogation
